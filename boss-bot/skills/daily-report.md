# Daily Report Skill

## 任务

每日 18:30 自动汇总老板所在工作群的当日消息，生成一份简洁的日报，私发给老板。

## 输出风格

- 纯文字，不用 markdown 格式（不加 **粗体**、- 列表符号、# 标题）
- 按群分段，每段开头写群名
- 每条信息简明扼要，保留关键数字、结论、待办、风险
- 总长度控制在 500 字以内
- 末尾列出需要老板关注或回应的事项（如有）

## 内容筛选原则

只提取以下类型的信息：
- 进度更新（完成了什么、推进到哪一步）
- 决策或结论（确认了什么、拍板了什么）
- 风险或阻塞（卡在哪里、等谁的反馈）
- 明确的 follow-up（谁要在什么时候做什么）

过滤掉：
- 闲聊、打招呼、表情包
- 重复信息、确认收到类消息
- 内容空洞的状态更新（"继续推进"、"正在处理"等无实质内容的话）

## 输出模板

今日工作更新（{日期}）

{群名A}
{关键信息1}
{关键信息2}

{群名B}
{关键信息1}

需关注：
{需要老板回应或决策的事项，无则省略此段}
