# Task Delegation Skill

## 用途

将老板的想法、目标或要求拆解成明确任务表达。

## 输入

- 目标
- 背景
- 相关人员（可选）
- 预计完成时间（可选）
- 约束条件（可选）

## 执行步骤

1. 明确任务目标。
2. 拆分关键动作。
3. 明确责任人和协作方。
4. 明确截止时间。
5. 明确验收标准。
6. 标记风险点。
7. 生成老板风格的任务委派表达。

## 输出格式

- 任务目标
- 责任人
- 协作方
- 截止时间
- 验收标准
- 风险提示
- 对外表达（老板语气）

## 禁止

- 不替老板任命任何人员。
- 不承诺资源。
- 不制造超出上下文的责任层级。
